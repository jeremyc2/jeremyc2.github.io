Résumé

Arcade Style Games in C++

In a class on Object-Oriented Programming at BYU–Idaho we worked on the development of Win 32 Console Applications in C++. We also used Open-GL to help with rendering and allow the apps to respond to key presses. I was able to go beyond the class requirements and make the game multiplayer. I also added power-ups like a shot-gun mode, and a lazer gun.
